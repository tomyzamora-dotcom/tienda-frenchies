# Tienda Web

Este proyecto es una tienda en línea simple creada con HTML, CSS y JavaScript.

## Archivos

- `index.html` - estructura de la página
- `style.css` - estilos visuales y diseño responsive
- `script.js` - lógica del carrito, almacenamiento y compra

## Cómo usar

1. Abre `index.html` en cualquier navegador moderno.
2. Explora los productos y haz clic en "Agregar".
3. Ajusta cantidades desde el carrito.
4. Haz clic en "Comprar" para finalizar.

## Funcionalidades

- Carrito persistente usando `localStorage`
- Cálculo automático del total
- Botones para aumentar/disminuir cantidad
- Alerta de compra exitosa

## Notas

- JavaScript debe estar habilitado para que la tienda funcione.
- Si el navegador no soporta `localStorage`, el carrito funcionará temporalmente mientras la página esté abierta.
